## Features

- **Knowledge Enhanced**: Uses RAG to answer questions about your codebase.
- **Interactive**: Built with Bubble Tea for a rich terminal UI.
- **Fast**: Uses Ollama for local LLM inference.

## Installation

1. Clone the repository:

```bash
git clone <repository-url>
cd kea
```

2. Install dependencies:

```bash
go mod tidy
```

3. Run the agent:

```bash
go run ./cmd/kea-cli
```
